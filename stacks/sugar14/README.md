# Sugar Dockerized Sugar version 14
This repository will help you deploy a Docker based development full stack for Sugar version 14

## Stacks available
* php83-local-build.yml - Main reference stack - Single web server
    * PHP - 8.3.6
    * Apache - 2.4
    * MySQL - 8.3.0
    * Elasticsearch - 8.4.3
    * Redis - latest
